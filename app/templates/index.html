Available Routes

All Countdown Event Routes

PATH METHOD REQUEST BODY DESCRIPTION
/coundowns GET N/A Get all countdown_events
/coundowns POST "title", "countdown_till_date" Create a countdown event
/coundowns/<id> GET N/A Get details for a countdown_event by ID
/countdowns/<id> PUT "title", "countdown_till_date" Update details for a countdown_event by ID
/countdowns/<id> DELETE N/A Delete a countdown_event by ID
